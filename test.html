While Loop — Basics

A while loop repeats as long as its condition is true. Three parts: init (before loop), condition (checked each time), advancing (moves toward ending).

// INIT
cnt = 0;
while (cnt < 100) {        // CONDITION
    cout << cnt << endl;   // BODY
    cnt = cnt + 2;         // ADVANCING
}

for loops = tend to be COUNTING loops. while loops = tend to be SENTINEL loops.

While Loop — Input Validation

Get the value first (init), then loop while it's invalid (condition), re-getting inside (advancing).

// Validate a number range
cout << "Enter your age: ";
cin >> age;                         // INIT
while (age <= 0 || age > 125) {    // CONDITION
    cout << "Invalid. Enter valid age: ";
    cin >> age;                     // ADVANCING
}

// Validate a char value
cin >> formatCode;
while (formatCode != 's' && formatCode != 'p') {
    cout << "Enter s or p: ";
    cin >> formatCode;
}

// Validate a positive number
cin >> price;
while (price <= 0) {
    cout << "Enter a positive value: ";
    cin >> price;
}

When validating a char, use char code; (no [80]) so != works correctly.

Sentinel Loops

A sentinel is a special value that signals "stop". The loop runs until it sees that value.

// Sentinel: 0 or negative quits
cout << "Enter number (0 quits): ";
cin >> num;               // INIT (priming read)
while (num > 0) {         // CONDITION
    dogYears = num * 7;
    cout << "Dog: " << dogYears << endl;
    cout << "Enter number (0 quits): ";
    cin >> num;           // ADVANCING
}

// Sentinel: read until file ends
while (fileVar) {         // true while file has data
    // process...
}

// Sentinel: user types Y/N to stop
done = 'N';               // INIT
while (done == 'N') {     // CONDITION
    // do work...
    cout << "Done? (Y/N): ";
    cin >> done;          // ADVANCING
}

The "read before the loop and also at the bottom" pattern is called a priming read.

Priming Read Pattern

When reading is the advancing mechanism, you must read ONCE before the loop starts, then again at the BOTTOM of the loop body.

// Read file until end — priming read pattern
GetSaleData(inF, desc, qty, price);  // BEFORE loop
while (inF) {
    ComputeExtended(ext, qty, price);
    DisplayLine(desc, qty, price, ext);
    GetSaleData(inF, desc, qty, price); // BOTTOM
}

// Sum odd numbers until even (sentinel = even)
inF >> num;               // priming read
while (num % 2 == 1) {    // while odd
    sum = sum + num;
    inF >> num;           // advancing at bottom
}

Nested While Loops

A while loop inside another while loop. Common pattern: outer loop does main work, inner loop validates input.

// Outer: keep going until user is done
// Inner: validate each input
done = 'N';
while (done == 'N') {

    // Get a valid positive number
    cout << "Enter positive number: ";
    cin >> num;
    while (num < 0) {          // inner validation
        cout << "ERROR - positive only: ";
        cin >> num;
    }

    // update even/odd counts
    if (num % 2 == 0)
        evenCnt = evenCnt + 1;
    else
        oddCnt = oddCnt + 1;

    // validate done answer
    cout << "Are you done? (Y/N): ";
    cin >> done;
    while (done != 'N' && done != 'Y') {
        cout << "Use Y or N only: ";
        cin >> done;
    }
}

Accumulation & Counting

// Accumulator — running total
total = 0.00;              // INIT before loop
while (...) {
    total = total + price; // ADD each time
}

// Counter — running count
count = 0;                 // INIT before loop
while (...) {
    count++;               // count = count + 1
}

// Conditional accumulation
if (code == 's')
    subTotal = subTotal + price;
else
    pkgTotal = pkgTotal + price;

Always initialize totals/counts to 0 BEFORE the loop — not inside it.

while(fileVar) — Reading Until End of File

When you don't know how many records are in a file, use a priming read + while(fileVar) instead of a counted for loop.

// Pattern 1: process file with known count (for loop)
dataFile >> numItems;
for (int i = 0; i < numItems; i++) {
    GetSaleData(dataFile, desc, qty, price);
    // process...
}

// Pattern 2: process file with UNKNOWN count (while loop + priming read)
totalCost = 0;
shipCount = 0;
GetShipment(inFile, weight, distance);     // priming read BEFORE loop
while (inFile) {                           // true while file has more data
    ComputeCost(weight, distance, ..., cost);
    totalCost = totalCost + cost;          // accumulate
    shipCount++;                           // count
    GetShipment(inFile, weight, distance); // advancing at BOTTOM
}
inFile.close();

The priming read must appear both BEFORE the loop and at the BOTTOM of the loop body. This ensures the condition is tested with fresh data each time.

Confirm Loop (Y/N re-entry pattern)

Used in ReadShippingRates — show all values, ask if correct, repeat if not.

correct = 'N';              // INIT
while (correct == 'N') {   // repeat until confirmed
    cout << "Enter base fee: ";
    cin >> baseFee;
    // ... get all values ...
    cout << "Is this correct? (Y/N): ";
    cin >> correct;
    cin.ignore(1000, '\n');
}

correct must be declared as char correct; — not char correct[80]. Single char variables compare with ==.

cin.ignore() Rule

  • >> followed by >> → no ignore needed
  • >> followed by getline → always ignore!
  • getline followed by anything → no ignore needed
cin >> num;
cin.ignore(1000, '\n');  // MUST before getline
cin.getline(name, 80);

// Multiple >> in a row: fine
cin >> price >> qty >> code;
cin.ignore(1000, '\n');  // only at the end

// Priming read + while(file) + accumulation
void ProcessEachShipment(char fileName[ ],
        double baseFee, double costPerPound,
        double costPerMile, double overWeightLimit,
        double overWeightSurcharge,
        double & totalCost, int & shipCount) {
    double weight, distance, cost;
    ifstream inFile;

    totalCost = 0;        // init BEFORE loop
    shipCount = 0;        // init BEFORE loop

    inFile.open(fileName);
    GetShipment(inFile, weight, distance); // PRIMING
    while (inFile) {
        ComputeCost(weight, distance,
            baseFee, costPerPound, costPerMile,
            overWeightLimit, overWeightSurcharge,
            cost);
        totalCost = totalCost + cost;      // accum
        shipCount++;                        // count
        GetShipment(inFile, weight,        // advance
            distance);
    }
    inFile.close();
}
