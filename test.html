Identifiers & Constants

  • Start with a letter, letters + digits only (underscore ok)
  • Case-sensitive: name ≠ Name
  • No reserved words. Must be descriptive.
  • camelCase vars: firstName
  • UPPER_CASE consts: NUM_PEOPLE
  • PascalCase funcs: GetData
// Named constant — declared GLOBALLY (above main)
const int STRING_WIDTH = 80;
const double TAX_RATE = 0.0875;

const must be declared globally, above main. Never use variables globally.

Arithmetic & Operators

Order (PEMD%AS): unary - → * / % → + - (left to right)

27 / 4  = 6     // integer division, truncates
27 % 4  = 3     // modulus = remainder
11 / 3  = 3     // NOT 3.67!
11 % 3  = 2

Rounding trick (add 0.5 before casting):

changeInCents = (int)(amount * 100 + 0.5);

Ceiling division (round UP without floats):

trucks = (qty + cap - 1) / cap;

Input / Output

// Input — leaves newline behind!

//       before a getline!

Char Arrays (Strings)

char name[80]; // holds up to 79 chars + '\0'
// null-terminated: good data ends with '\0'
// '\0' is char 0 — NOT the same as '0' (char 48)!
// Read whole line (spaces included)
cin.getline(name, 80);
fileVar.getline(name, 80);

// Read one word (stops at space)
cin >> name;
fileVar >> name;
// Iterating through a char array
for (int i = 0; i < 80 && name[i] != '\0'; i++)
    cout << name[i];

Functions

// 1. PROTOTYPE (declaration) — above main, describes the function
void ShowLine(char [80], double);       // void = returns nothing
void GetData(double &);                 // & = pass by reference
int  ComputeAvg(int, int, int, int, int, int, int);  // returns int

// 2. CALL — inside main
GetData(amountToAdd);                   // reference: changes the original
ShowLine(name, grade);                  // value: uses a copy
avg = ComputeAvg(t1,t2,t3,t4,t5,t6,t7);
cout << ComputeAvg(t1,t2,t3,t4,t5,t6,t7) - 5 << endl; // return value in expression

// 3. DEFINITION — after main
void GetData(double & amountToAdd) {    // & modifies the ORIGINAL variable
    cout << "Enter amount: ";
    cin >> amountToAdd;
    cin.ignore(1000, '\n');
}
int ComputeAvg(int t1, int t2, int t3, int t4, int t5, int t6, int t7) {
    int average;
    average = (t1+t2+t3+t4+t5+t6+t7) / 7;
    return average;             // sends value BACK to caller
}
  • Pass by value (double x) — makes a copy, original unchanged
  • Pass by reference (double & x) — shares memory, original IS changed
  • ifstream / ofstream must always be by reference: (ifstream & f)
  • Local variables (under the heading) only visible inside that function
  • ALL variables must be defined in the function — under the heading OR passed in as parameters

Program Layout — Apply to Any Question

// ── STEP 1: INCLUDES (only what you need) ─────────────────────
#include <iostream>   // cin, cout, endl
#include <fstream>    // ifstream, ofstream
#include <iomanip>    // setw, setprecision, fixed, left, right, showpoint
using namespace std;

// ── STEP 2: HEADER COMMENT ────────────────────────────────────
// Programmer:        Mattox Ward
// Date Due:          Friday, April 25, 2026
// Project Number:    9
// Project Name:      Laboratory09
// Project Description: One or two sentences describing
//                      what the program does.

// ── STEP 3: GLOBAL CONSTANTS (never variables here!) ──────────
const int STRING_WIDTH = 80;
const double TAX_RATE = 0.0875;

// ── STEP 4: FUNCTION PROTOTYPES (if using functions) ──────────
void GetFileName(char [80]);
void GetData(ifstream &, char [80], char [80], double &, int &);
void DisplayResults(char [80], double, int);

// ── STEP 5: MAIN FUNCTION ─────────────────────────────────────
int main(void) {

    // 5a. Define ALL local variables at the top
    char fileName[80], custName[80];
    double price, total;
    int quantity;
    ifstream dataFile;

    // 5b. Get input (ask user for filename, read rest from file)
    GetFileName(fileName);
    GetData(dataFile, fileName, custName, price, quantity);

    // 5c. Calculate
    total = price * quantity * (1.0 + TAX_RATE);

    // 5d. Display output
    DisplayResults(custName, total, quantity);

    return 0;   // always last line of main
}

// ── STEP 6: FUNCTION DEFINITIONS (after main) ─────────────────
void GetFileName(char fileName[80]) {
    cout << "Enter file name: ";
    cin >> fileName;
    cin.ignore(1000, '\n');
}
void GetData(ifstream & dataFile, char fileName[80],
        char custName[80], double & price, int & quantity) {
    dataFile.open(fileName);
    dataFile.getline(custName, 80);
    dataFile >> price >> quantity;
    dataFile.close();
}
void DisplayResults(char custName[80], double total, int quantity) {
    cout << fixed << setprecision(2);
    cout << left << setw(20) << custName
         << right << setw(10) << total << endl;
}
